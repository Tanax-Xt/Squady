### Процесс регистрации

Процесс регистрации в Squady состоит из следующих пунктов:

1. Перейти на страницу регистрации
2. Ввести необходимые данные: логин, email, пароль
3. Подтвердить электронную почту с помощью одноразового кода

Далее пользователю открывается доступ к процессу онбординга, который описан в следующем разделе
[Аккаунт пользователя](#аккаунт-пользователя).

## Аккаунт пользователя

> Аккаунт — ключ пользователя на платформу, с его помощью можно создавать резюме, искать команды и участвовать
> в мероприятиях.

### Онбординг

> Онбординг — процесс знакомства и первого соприкосновения со Squady, вам будет предложено выбрать роль
> и, для ролей `Участник` и `Наставник`, ввести личные данные — основу будущих резюме.

Информация об онбординге будет доступна в сводке на главной странице.

<Button variant="outline" className="my-4 w-fit" asChild>
  <Link href="/home" target="_blank">
    На главную
    <ArrowRightIcon />
  </Link>
</Button>

После регистрации и подтверждения аккаунта вам будет предложено выбрать одну из трех ролей:

- Участник — тот, кто ищет команду для участния в олимпиадах и хакатонах
- Наставник — помощник команды, который хочет делиться опытом
- Представитель олимпиады — организатор мероприятий (для роли необходима верификация через администрацию сайта)

Далее для ролей `Участник` и `Наставник` будет предложено ввести личные данные — основу будущих резюме. Эта информация
будет одинакова для всех резюме и упростит их создание. Подробнее в разделе [Личные данные](#личные-данные).

Следующий этап онбординга — создание резюме. Поддерживается ручное создание или импорт из различных источников.

## Резюме

> Резюме — визитная карточка пользователя, именно с её помощью происходит рекрутинг в команды.

Перед добавлением резюме нужно указать [Личные данные](#личные-данные). Они будут идентичны для всех резюме.

В резюме пользователь может указать следующие данные:

- Роль, будет отображаться в названии резюме. Может быть только одно резюме с каждой уникальной ролью
- Учебное заведение
- Навыки
- Опыт, здесь можно указывать как учебный, так и рабочий опыт
- Достижения
- Дополнительное образование
- Настройка публичности резюме, нужно, чтобы резюме выдавалось при поиске лидером команды. Подробнее
  в [Приглашение в команду](#приглашение-в-команду)

Работать с резюме можно из раздела [Мои резюме](/resumes) или сайдбара с правой стороны сайта.

### Личные данные

> Личные данные — информация об участнике, которая заполняется во время онбординга и содержит в себе данные,
> которые будут идентичны для всех резюме пользователя.

Личные данные включат в себя:

- ФИО пользователя
- Дату рождения
- Город проживания
- Телеграм для связи
- Поле "О себе"

Отредактировать личные данные можно в разделе [Мои резюме](/resumes) во вкладке
[Мои личные данные](/resumes/profile).

### Ручное создание резюме

В Squady пользователю доступно несколько способов создания резюме: ручное и импорт из PDF-файла,
с GitHub или HeadHunter. Подробнее про импорт резюме в разделе [Импорт резюме](#импорт-резюме).

Для ручного создание резюме пользователю необходимо в разделе [Мои резюме](/resumes) нажать на кнопку "Создать"
и выбрать создание резюме с нуля или в сайдбаре в левой части сайта нажать на `+` в секции Мои резюме.

Далее пользователю будет предложено указать следующие данные:

- Роль, будет отображаться в названии резюме
- Учебное заведение
- Навыки
- Опыт
- Достижения
- Дополнительное образование
- Публичность резюме

После добавления всех данных пользователь может сохранить резюме. Редактирование резюме описано
в разделе [Действия с резюме](#действия-с-резюме)

### Импорт резюме

Для пользователей доступен импорт резюме из трех источников:

- PDF-файл, будет обработан ИИ. Импорт занимает около 20 секунд
- Профиль на GitHub
- Резюме с HeadHunter, необходимо, чтобы резюме было доступно по ссылке

После импорта резюме из любого источника пользователю будет доступен результат импорта и черновой вариант резюме,
который можно дополнительно подредактировать и сохранить.

### Действия с резюме

В разделе [Мои резюме](/resumes) пользователю доступны просмотр [Личных данных](#личные-данные) и всех его резюме.

Для каждого резюме доступны следующие действия:

- Открыть для подробного просмотра
- Скопировать ссылку на резюме
- Редактирование
- Смена режима приватности (публичное/приватное)
- Дублирование резюме: для тех случаев, когда хочется сделать новое резюме на основе старого
- Удаление

Редактирование резюме аналогично процессу [Ручного создания резюме](#ручное-создание-резюме) и открывает возможность
для обновления, добавления или удаления информации из резюме.

## Команды

> Команды - группы участников до 10 человек, которые участвуют в одном мероприятии. В составе команды могут быть
> участники с ролями `Участник` и `Наставник`.

Пользователь вступает в команду с конкретным резюме. Это сделано для того, чтобы было четкое распределение ролей и зон
ответственности в командах.

### Создание команды

Создание команды доступно только пользователям с ролями `Участник` и `Наставник`.

Для создание мероприятия пользователю необходимо в разделе [Команды](/teams) нажать на кнопку "Создать" или в сайдбаре
в левой части сайта нажать на `+` в секции Мои команды.

Далее пользователю будет предложено указать следующие данные:

- Название
- Описание
- Задачи
- Резюме, с которым он будет участвовать

После добавления всех данных пользователь может создать команду. Редактирование команды описано
в разделе [Действия с командами](#действия-с-командами).

Пользователь, создавший команду, будет назначен ее лидом.

### Приглашение в команду

Лид команды может самостоятельно подбирать себе участников в команду. Для этого есть несколько способов: приглашение
по ссылке, по электронной почте и по резюме.

Для приглашения по ссылке или электронной почте лиду необходимо со страницы команды перейти в `Управление командой`,
в блоке `Приглашения` скопировать ссылку на вступление в команду и самостоятельно отправить ее или ввести
электронную почту участника, которому он хочет отправить приглашение на подачу заявки в команду.

Для того, чтобы пригласить пользователя по резюме, необходимо на странице команды перейти по кнопке `Подбор участников`.

Лиду откроется страница с публичными резюме других участников. Для того, чтобы лиду было проще найти подходящих
участников на платформе доступны удобные фильтры:

- По навыкам в резюме
- По уровню образования
- По количеству лет опыта в резюме
- По количеству проектов в резюме
- По роли в резюме

### Заявки в команду

Заявки на вступление в команду, которые пользователи подают из поиска по командом или по приглашению лида команды,
доступны для просмотра лиду на странице команды в разделе `Управление командой` в блоке `Заявки`. Лид может принять
или отклонить заявку на вступление пользователя.

Также на разделе `Управление командой` лиду доступна аналитика по заявкам в команду на определенное время.

Пример аналитики по заявкам за определенный период, доступен экспорт данных в csv.

### Действия с командами

Просмотр команд, в которых состоит пользователь, доступен на [Домашней странице](/home), на странице [Команды](/teams)
и в сайдбаре в левой части экрана.

Пользователь, который не имеет роль лида в команде, может посмотреть резюме других участников команды и покинуть команду.
Также участник команды может скопировать ссылки на команду и подачу заявки на вступление в команду.

Лиду команды доступны следующие действия:

- Копирование ссылки на команду
- Копирование ссылки на подачу заявки в команду
- Редактирование команды
- Управление заявками в команду, подробнее в разделе [Заявки в команду](#заявки-в-команду)
- Расформирование команды

Редактирование команды аналогично процессу [Создания](#создание-команды) и открывает возможность
для обновления, добавления или удаления информации о команде.

## Мероприятия

> Мероприятия — интеллектуальные соревнования, которые создают верифицированные представители. Участники и наставники
> с легкостью могут найти интересующее их событие.

### Создание мероприятий

Создание мероприятий доступно только верефицированным пользователям с ролью `Представитель олимпиады`.

Для создание мероприятия пользователю необходимо в разделе [Мероприятия](/events) нажать на кнопку "Создать".

Далее пользователю будет предложено указать следующие данные:

- Название
- Описание
- Даты проведения
- Локацию
- Формат события (оффлайн/онлайн/гибрид)

После добавления всех данных пользователь может сохранить мероприятие. Редактирование мероприятия описано
в разделе [Действия с мероприятиями](#действия-с-мероприятиями).

### Действия с мероприятиями

Верефицированным пользователям с ролью `Представитель олимпиады` доступны следующие действия с их мероприятиями:

- Копирование ссылки на событие
- Редактирование
- Удаление

Для этого из раздела [Мероприятия](/events) необходимо перейти на страницу конкретного события и выбрать действие
в меню в правом верхнем углу.

Редактирование мероприятия аналогично процессу [Создания](#создание-мероприятий) и открывает возможность
для обновления, добавления или удаления информации о события.
